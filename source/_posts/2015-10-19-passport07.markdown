####排他的論理和(XOR)
真理値表  

| 命題P   | 命題Q   | P XOR Q |
|--------|--------|--------|
|  TRUE  |  TRUE |    FALSE    |
|  TRUE  |  FALSE |    TRUE    |
|  FALSE  |  TRUE |    TRUE    |
|  FALSE  |  FALSE |    FALSE    |

どちらか片方のみが真であれば、その排他的論理和も真であるが、両方が真だと偽になる。
